PROJECTS

Here you can find a list of interesting projects I've done over the years.

Firmware/Hardware projects:

AC Signal Capturer

This is a C program implemented on an EFM8 system to analyze two AC signals and return the RMS voltages, periods, and phase differences on a backlit display.

Baccarat

This is an implementation of the classic board game Baccarat using SystemVerilog HDL synthesized on a DE1-SoC FPGA board. It uses an internal clock-based algorithm to emulate the dealt cards.

Water Level Measurer

This is an tool that audibly outputs the amount of water in a plastic cup by using the dielectric properties of liquids. This was built on a 8051-based MCU and programmed using Assembly.

Emb. RTOS

This was my first implementation of an RTOS on an embedded system (STM32F411). I used C and ARM assembly to program the kernel and included support for safe concurrency.

Software projects:

Document Analyzer

This is a text analyzer that produces quantitative measurements based on specific text metrics and criteria. It accepts both text files and webpages as input.

WikiMediator

This is a wrapper made for the jwiki API which records Wikipedia queries made by a user. It supports multithreading and is built to be thread-safe.

Malware Modelling

This project simulates malware sent from a user into a network, keeps track of infection rates, and can determine the optimal way for malware propagation.

OISC Machine

This project simulates OISC CPU architecture and produces the memory map needed to perform certain arthimetic operations.
