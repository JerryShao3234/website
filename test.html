PROJECTS

DNA Comparator

This is a C program that takes in a DNA sequence as a string and determines the similarity between each candidate and a set of nucleotides. It follows a specific rubric to determine this quantifiable score.

Baccarat

This is an implementation of the classic board game Baccarat using SystemVerilog HDL synthesized on a DE1-SoC FPGA board.

Water Level Measurer

This is an overengineered tool that determines the amount of water in a plastic cup by using the dielectric properties of liquids. This was built on a 8051-based MCU and programmed using Assembly.
